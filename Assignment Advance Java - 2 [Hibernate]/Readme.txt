This is the project created during the training session at Nagarro. Assignment-2 of Advance Java is what to be followed in this project.

src folder contains the source files and helpingFiles folder has the xml files used to create this project.

helpingFiles->
POM.xml has all the dependencies used in the Maven project.
hibernate.cfg.xml has the hibernate properties used.

src->
App.java is the main file behind the project.
thread_ImportData.java file is used to read multiple files in the folder.
createDB.java is using the thread file to create the database with data stored in the csv files.
readDB.java file is used to read data from the database.
flightdata.java is the helper file while is helping App.java to read and store data.
